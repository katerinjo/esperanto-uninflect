# espo-uninflect

Uninflect Esperanto words found in the wild to their dictionary forms for easy lookup. Both standard Espenanto orthography and the x system are recognized.

## Examples

- "kial" -> "kial"
- "bona" -> "bona"
- "studantoj" -> "studanto"
- "homojn" -> "homo"
- "sunplenan" -> "sunplena"
- "hejmen" -> "hejme"
- "amis" -> "ami"

Unofficial words and affixes are also handled.

- "naj" -> "na"
- "rian" -> "ri"
- "alies" -> "alies"
